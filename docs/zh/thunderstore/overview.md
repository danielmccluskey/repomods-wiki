# Thunderstore 概览

### 什么是 Thunderstore？
R.E.P.O. 模组社区已将模组的托管和分发完全集中在 **Thunderstore.io** 上。社区选择 Thunderstore 是因为它与 Thunderstore App、Gale Mod Manager 和 r2modman 等模组管理器无缝集成，允许玩家：

* 轻松安装模组。
* 通过配置代码与朋友同步相同的模组设置。
* 自动解析技术依赖。

### 在 Thunderstore 上创建团队
如果你想发布自己的 R.E.P.O. 模组，Thunderstore 要求你创建一个 **"团队"**，它作为你发布模组的永久开发者命名空间。

#### 如何设置：
1.  前往 [thunderstore.io](https://thunderstore.io) 并点击右上角的 **"Login with..."**，使用 **GitHub、Discord 或 Overwolf** 账户登录。
2.  点击窗口右上角的 **个人资料名称** 以打开账户设置。
3.  导航到 **"Teams"** 选项卡。
4.  点击 **"Create Team"** 按钮。
5.  输入你想要的团队名称（如果不是团队协作，建议使用你的标准用户名）。

> [!CAUTION]
> **严重警告：** 一旦你在一个特定的团队下上传了模组，你的 Thunderstore 团队将无法删除或重命名，因此请谨慎选择名称！
