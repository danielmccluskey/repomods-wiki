# MenuLib 概览

MenuLib 是一个编程 API，旨在简化 R.E.P.O. 中自定义 UI 菜单的创建和管理。

::: warning 开发中
此文档页面目前是一个占位符。详细的 API 参考和使用示例将很快添加。
:::

### 功能特点
- **轻松的菜单注册**：快速向游戏现有 UI 添加自定义按钮和面板。
- **标准化 UI 元素**：使用与游戏美学匹配的预构建组件。
- **事件处理**：简单的按钮点击和值更改钩子。

### 快速开始
要在你的项目中使用 MenuLib，请确保你已按照 [HarmonyX 项目设置](../../harmonyx.md#adding-thunderstore-dependencies) 中的描述将依赖添加到你的 `.csproj`。
