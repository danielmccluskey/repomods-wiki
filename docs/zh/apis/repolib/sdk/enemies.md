# 使用 REPOLib-Sdk 创建敌人

::: warning 注意
本页面目前仍在开发中！
:::

::: info 注意
**本指南假设你已经为 REPOLib modding 设置好了 REPOLib Unity 项目。\
如果尚未设置，请先参阅[快速入门](./start.md)。**
:::

- 创建一个 `EnemySetup`，右键点击并选择 `Create > Other > Enemy Setup`。
  - 与 Valuables 一样，可以参考原版内容。
- 在你的 mod 文件夹（或任何子文件夹）中右键点击，选择 `Create > REPOLib > Enemy`。
- 填写各个字段。
