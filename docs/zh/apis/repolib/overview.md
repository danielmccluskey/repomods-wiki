# REPOLib 概览

REPOLib 是一个广泛使用的库，用于为 R.E.P.O. 添加内容。

- [Thunderstore](https://thunderstore.io/c/repo/p/Zehs/REPOLib/)
- [GitHub 仓库](https://github.com/ZehsTeam/REPOLib)

## 使用方式

使用 REPOLib 有两种主要方式：

- [**通过 C# API。**](./api/start.md) 最灵活的方式，但需要编写代码。
- [**通过 Unity 编辑器。**](./sdk/start.md) 不需要编写代码，但灵活性较低。

选择哪种方式取决于你的经验水平和项目规模。\
对于新手 modder 和小型项目，推荐使用 Unity 工作流。
