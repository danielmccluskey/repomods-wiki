# R.E.P.O. 概述

**R.E.P.O.** ([Steam 链接](https://store.steampowered.com/app/3241660/REPO/)) 是一款在线合作恐怖游戏，最多支持六名玩家合作寻找和提取具有物理模拟效果的贵重物品。玩家必须小心处理这些物品，将它们取出并满足创造者的需求。

## 为 R.E.P.O. 安装 Mod

通过安装 Mod 来增强你的 R.E.P.O. 体验。请按照以下步骤开始：

1. **获取 Mod 管理器**：
目前有三个可用于 R.E.P.O. 的 Mod 管理器。
- [Gale](https://thunderstore.io/package/Kesomannen/GaleModManager/)（用户友好界面，支持 Windows 和 Linux）
- [r2modman](https://thunderstore.io/package/ebkr/r2modman/)（类似 Thunderstore Mod Manager 的界面，支持 Windows 和 Linux）
- [Thunderstore Mod Manager](https://www.overwolf.com/app/thunderstore-thunderstore_mod_manager)（仅支持 Windows，需要 Overwolf）
   
**我们推荐使用 Gale 或 r2modman。**

::: danger 手动安装 Mod
**强烈不推荐手动安装 Mod！**

**在我们的社区 Discord 服务器中，手动安装将不会获得任何支持！**
:::

2. **安装并打开你的 Mod 管理器**
下载、安装并启动你选择的 Mod 管理器。

3. **选择 R.E.P.O. 并安装 Mod**
在 Mod 管理器中选择 **R.E.P.O.**，然后浏览并安装你想要的 Mod。

4. **启动并开始游戏**
通过 Mod 管理器启动 **R.E.P.O.** 以加载你的 Mod 并享受游戏。

## 游戏中遇到问题？
在使用 Mod 时遇到卡在加载界面、随机断线或崩溃等问题？\
请前往我们的 [Discord 社区服务器](https://discord.gg/vPJtKhYAFe) 的 [`#mod-support`](https://discord.com/channels/1344557689979670578/1347085657864015872) 频道寻求帮助。
